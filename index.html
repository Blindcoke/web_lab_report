Предметне середовище: Предметним середовищем для лабораторної роботи є розробка веб-сайту для пекарні. Для цього використовується сучасний стек технологій, включаючи HTML, CSS і JavaScript для розробки фронтенду. Для управління версіями та спільної роботи над проєктом використовується система контролю версій Git та хмарний сервіс GitHub.

Способи підключення стилів

Існує кілька способів підключення стилів до веб-сторінки: Вбудований CSS: Стилі можна вбудувати безпосередньо всередині HTML-документу, використовуючи тег <<style>> в розділі <<head>>. Зовнішній CSS: Стилі можна розмістити в окремому файлі CSS з розширенням .css, а потім підключити його до HTML-сторінки за допомогою тегу <<link>>. Inline CSS: Можна також задати стилі безпосередньо всередині тегів HTML за допомогою атрибуту style.
